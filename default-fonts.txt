Configuration to change default fonts in linux.
1.-create this file at '/home/bryant/.config/fontconfig/fonts.conf'
2.-Copy this content into the previous file 'fonts.conf'
<?xml version='1.0'?>
<!DOCTYPE fontconfig SYSTEM 'fonts.dtd'>
